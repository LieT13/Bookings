Welcome to Bookings!

Users management endpoints:

  • GET all users: {{host}}/Bookings/rest/users
  • GET a user by id: {{host}}/Bookings/rest/users/{userId}
  • POST (create) a user: {{host}}/Bookings/rest/users
  • PUT (update) a user: {{host}}/Bookings/rest/users/{userId}
  • DELETE a user: {{host}}/Bookings/rest/users/{userId}
